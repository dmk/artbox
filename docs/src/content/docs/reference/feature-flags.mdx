---
title: Feature Flags
description: Optional features and what they enable.
---

## images

Enables:
- Terminal image output (kitty/iTerm2)
- ASCII image rendering
- `Artbox::render_image_path` / `render_image_bytes`

```toml
artbox = { version = "0.2", features = ["images"] }
```

## cli

Enables:
- The `artbox` binary
- CLI argument parsing for examples like `gradient`

The `artbox` binary itself requires both `cli` and `images`.

```toml
artbox = { version = "0.2", features = ["cli", "images"] }
```

## ratatui

Enables:
- `integrations::ratatui::ArtBox` widget
- `integrations::ratatui::SpriteBox` widget

```toml
artbox = { version = "0.2", features = ["ratatui"] }
```
